前端 Link to heading

    网页的结构，组件的位置 - 骨骼

  • CSS

    网页的美化，组件的style - 服饰

  • JavaScript

    网页的交互，组件的交互逻辑 - 逻辑

JavaScript Link to heading

你们人类不满足网页只是一些文字和图片的展示，希望页面上可以有更多功能。
比如点击一个按钮，弹一个窗口 或者改变页面上某些内容。 为了实现这种功能，就创造了一门脚本语言，逐步升级演化成了 JavaScript 这门语言。

JavaScript 框架 Link to heading

  • jQuery

    解决浏览器之间的兼容性问题

  • Vue

    通过尽可能简单的API实现响应的数据绑定和组合的视图组件

  • AngularJS

    把后台的json值直接用html进行渲染，然后html的操作又直接在形成json传回后台。

  • React

    React是用于渲染UI的JS库，定位在与实现UI React Web App解决方案，衍生的React Native是跨屏App解决方案

Node.js Link to heading

node.js 是一个基于 Chrome V8 引擎的 JavaScirpt 运行环境。能让JavaScirpt不仅仅只能运行在浏览器中，通过浏览器调用服务。而且能让JavaScirpt直接调用服务

AJAX Link to heading

AJAX是一种技术，直白地说，就是没用AJAX的网页，你点一个按钮就要刷新一下页面，尽管新页面上只有一行字和当前页面不一样， 但你还是要无聊地等待页面刷新。用了AJAX之后，你点击，然后页面上的一行字就变化了，页面本身不用刷。

JSON Link to heading

为了描述复杂的数据，一开始是用一些字符分割拼起来，后来人类用xml来描述，发现解析还是挺麻烦的。
于是就基于JavaScript的数据类型创造了JSON,用来描述前后端数据交互的内容格式，JSON属于JavaScript的一个子集
