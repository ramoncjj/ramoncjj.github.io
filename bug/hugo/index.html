@ECHO OFF
hugo new site demo
cd demo
cd themes
# 把hugo-coder主题下载到themes里
git clone https://github.com/luizdepra/hugo-coder.git
cd hugo-coder
cd exampleSite
move config.toml ../
cd..
move config.toml ../
cd..
move config.toml ../
cd..
# 开启本地预览
start http://localhost:1313/ 
# 启动
hugo server

PAUSE
# 复制代码保存为bat文件，双击执行

到上面为止能开启本地预览了

部署github Link to heading

  1. 在github上创建仓库：[你的 github username].github.io（必须是这个仓库名字）
@ECHO OFF
# 进入刚才的dem
cd demo
# 在public中生成静态文件
hugo 
cd public
git init
git add .
git commit -m "first commit"
git remote add origin [刚刚新建的仓库地址]
git branch -M main
# 将public中的生成的静态文件上传到刚刚新建的仓库
git push -u origin main
PAUSE
# 复制代码保存为bat文件，双击执行

  1. 在github上刚才的仓库中

    setting -> pages source: Deploy from a branch Branch: main -> save

  2. 部署完毕，访问 https://[你的 github username].github.io/
